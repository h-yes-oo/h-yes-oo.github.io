내가 개발을 시작하려는 이유

각각의 관심도에 맞는 개발 공부 방향 정하기

  • 소프트웨어적으로 무언가 만들어보고 싶은 것이 생겼기 때문에

    • 프레임워크에 대해 더 공부하기 !
      1. JAVA - Spring
      2. Python - django
      3. Javascript - React

  • 개발이라는 것에 관심이 생겼고, 개발자로 살아보고 싶어요

    • 책 : 소프트웨어 장인
    • 인터넷 강의 : 생활코딩

내가 개발을 시작하려는 이유는 우선 만들고 싶은 서비스가 있어서 !!! 그리고 의미있는 서비스를 창출해내는 기업에 속해서 내 몫의 무언가를 하고 싶어서 / 개발을 할 줄 안다는 건 의미 창출에 어떻게든 기여할 수 있을 거라는 생각에

개발 분야 정하기

Client <-> Server <-> DataBase

  • 프론트엔드 개발자 : Client 를 개발하면서, Server로 어떻게 요청을 보내는지 알아야해요
  • 백엔드 개발자 : Server를 만들지만, Database에 자료를 어떻게 넣을지까지 고민해야해요
  • 데이터 사이언티스트 : 쌓여진 DataBase를 다루는 일을 해요

우선은 넓게 다 경험해본 이후에, 세부 분야 올해 말까지 정할 것 ! 풀스택 개발자를 지향하기
