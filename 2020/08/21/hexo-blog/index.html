SEO 적용하기

SEO 란 ?
SEO는 Search Engine Optimization의 약자로, 우리말로는 검색 엔진 최적화라고 합니다 !
구글, 네이버 등의 검색엔진이 자료를 수집하고 순위를 매기는 방식에 맞도록 웹페이지를 구성하는 작업을 의미합니다.
이 작업을 거치면 검색 엔진에서 검색 결과의 상위에 노출될 수 있습니다.
내가 쓴 글이 누군가의 필요에 의해 읽히기를 원하신다면 SEO 작업을 진행해보세요 ٩( ‘ω’ )و

hexo-auto-canonical

hexo-auto-canonical 은 블로그의 대표 url을 검색엔진에 알려주는 url 표준화를 담당하는 플러그인입니다.

다음 명령어로 플러그인을 먼저 설치해주세요.

npm install hexo-auto-canonical --save

현재 블로그에 적용된 테마 폴더 아래에 layout 폴더에 잘 찾아보시면 head.ejs 파일이 있습니다.

저는 tranquilpeak 테마를 적용하고 있는데 themes/tranquilpeak/layout/_partial/head.ejs 여기에 있네요 !

테마에 따라서 layout/common 폴더에 head.ejs 파일이 있다고도 합니다

찾으신 head.ejs 파일에
<%- autoCanonical(config, page) %>
요 코드를 삽입해 주세요 !
